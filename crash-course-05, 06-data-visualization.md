# Data Visualization

## Quantitative Data

- has both order and consistent spacing
- ex perchases of olive oil by ounce

## Categorical Data

- does not have meaningful order or consisent spacing
- ex favorite pastas (macaroni, spaghetti, angel hair, rotini)

## Frequency table

Most common for categorical data

|pasta type|frequency|relative frequency|
|---|---|---|
|macaroni|10|20%|
|linguine|15|30%|
|rotini|20|40%|
|penne|5|10%|

## Binning

takes a quantitative variable and bins it into categories (real or made up)

## Bar charts

represents quantities as comparable bars

## Histogram

a bar chart smooshed together to show continuous data

## Dot Plot

dots arranged in a line similar to a histogram

## Stem & Leaf plot

- stacks data on top of each other
    1. split each data into a stem and leave
        - stems are related to the bins or bars of data
    1. each datum is represented by the number minus the stem value
    1. if each stem counts for 10, 4 represents 14 for the datum

## Box Plots

- aka box & whisker plot
- visually display centra tendency and spread

```
[]: interquartile data q1 to q3
* : min/maximum
| : median at q2
- : whiskers show the min/max of data
{}: fence show 1.5x the distance of q1 to q3
+ : outlier

{        *-----[    |  ]--*     } +
```

- if the median is more distance from one side of the box, the quartile is more spread out
- data outside the fence is a potential outlier
- outliers are not always unrepressentative
- a valid but rare outlier *is still* representative

## Culmulative Frequency Plots

- each bar or point contains the bins that come before it.
