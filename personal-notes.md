# Statistics Personal Observations

## Central Tendency

Mean, Median, Mode are all methods to find the 'average' in some way. 

## Relationships

- $ P(A \cup B) $ = everything among A and B minus duplicates

- $ P(A \cap B) $ = values shared by A and B

- $ P(A | B) $ = Indicates a change in reference (denominator)

### Translation Table: Math-to-English

| Formal/Book Language                                                                 | Plain English                                             |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------|
| $P(A \cup B)$ = Probability that A or B or both occur (union of A and B)             | $P(A \cup B)$ = everything among A and B minus duplicates |
| $P(A \cap B)$ = Probability that both A and B occur (intersection of A and B)        | $P(A \cap B)$ = values shared by A and B                  |
| $P(A \mid B)$ = Probability of A given B; restrict sample space to B                 | $P(A \mid B)$ = change in reference (denominator)         |
| Conditional probability: denominator is the total for the given condition            | “GIVEN X” means use X’s total as denominator              |
| Marginal probability: denominator is the grand total unless a condition is specified | Use grand total unless “GIVEN” is specified               |
